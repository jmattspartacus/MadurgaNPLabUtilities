LevelSchemeUTK.jl

-----------------------------
This program draws any nuclear level scheme using the Plots.jl package for the Julia programming language.
-----------------------------

DEPENDENCIES
-----------------------------
Julia 1.6.2
Plots.jl
-----------------------------

Installation
-----------------------------
git clone https://github.com/mmadurga/Utilities/blob/master/LevelScheme
-----------------------------

Usage
-----------------------------
The level scheme is drawn from the contents of the levels[] and transitions[] arrays. 
All columns in both arrays must be filled to maintain the correct dimentions. Use empty quotes ("  ") for empty column values

Once all transitions and levels are specified simply run it from Julia's REPL.
