plotGates.hpprgm

-----------------------------
This program draws the trigger windows, VANDLE global, and Beta global validation windows. Trigger times are hardwired to 0, 10 and -100 ns for V1, V2 and beta respectively.
-----------------------------

DEPENDENCIES
-----------------------------
HP connectivity kit
-----------------------------

Installation
-----------------------------
git clone https://github.com/mmadurga/Utilities/HP \Prime/blob/master/plotGates.hpprgm and
Copy directly into the calculator using HP Connectivity Kit.
-----------------------------

Usage
-----------------------------
Gates are drawn from the gate values and delays,

PLOT(vandleGate,betaGate,globalValidation,triggerValidationdDelay)
